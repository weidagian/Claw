## 核心功能：批量生成多空间效果图

### 功能说明

用户上传一张户型平面图后，系统可以**一次性生成多个空间**的效果图，如客厅、主卧、次卧、餐厅等。所有效果图保持**风格统一**，但每个空间的内容根据其功能自动适配。

### 支持的空间类型

| 空间 | 图标 | 说明 |
|-----|------|------|
| 客厅 | 🛋️ | 沙发、电视背景墙、茶几、灯具 |
| 主卧 | 🛏️ | 床、床头柜、衣柜、灯具 |
| 次卧 | 🛏️ | 床、衣柜、书桌 |
| 餐厅 | 🍽️ | 餐桌餐椅、餐边柜、灯具 |
| 厨房 | 🍳 | 橱柜、灶台、操作台 |
| 卫生间 | 🚿 | 马桶、洗手台、淋浴 |
| 书房 | 📚 | 书桌、书柜、座椅 |
| 阳台 | 🌿 | 休闲座椅、绿植 |
| 玄关 | 🚪 | 鞋柜、换鞋凳 |
| 儿童房 | 🧸 | 儿童床、书桌、玩具收纳 |

### 快速选择预设

| 预设 | 包含空间 | 适用户型 |
|-----|---------|---------|
| 一室一厅 | 客厅 + 卧室 | 小户型 |
| 两室一厅 | 客厅 + 主卧 + 次卧 | 刚需户型 |
| 三室一厅 | 客厅 + 主卧 + 次卧 + 餐厅 | 改善户型 |
| 三室两厅 | 客厅 + 主卧 + 次卧 + 餐厅 + 厨房 | 改善户型 |
| 四室两厅 | 客厅 + 主卧 + 次卧 + 餐厅 + 厨房 + 书房 | 大户型 |

**默认选中：两室一厅**（符合大多数刚需用户）
